# About
A web application where users can hone and test their typing speed by typing quotes from famous authors.
# Features
- Featured authors include: Friedrich Nietzsche, Fyodor Dostoevsky, William Shakespeare and more
- The test can be customized by selecting a particular author and adjusting the time limit
- Users can track their scores and view their ranking against other users on the global leaderboard
- Email verification and google sign-in
